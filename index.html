Личная информация

Привет! Меня зовут Эмиль. Мне 22 года. Занимаюсь вёрсткой сайтов. Могу сверстать с PSD макета, а также редактировать уже готовые проекты по желанию заказчика. Взятые заказы сдаются строго в обещанные сроки.

Знания

Владения такими языками как HTML, CSS, Javascript (+J-Query), PHP, а также СУБД MySQL. Способность работать с фреймворком Bootstrap и технологией FlexBox. Oпыт работы с графическим редактором PhotoShop CC.

Резюме

С более обширной информацией обо мне вы можете ознакомится в моем резюме, кликнув по значку загрузки.

Используюмые
технологии

HTML5 + CSS3

Современная верстка по макету. Семантичный код без использования однообразных тегов.

Чистый код

Чистый, комментированный код, понятный любому программисту. Все идентификаторы и классы получают название по методологии БЭМ

Валидность

Перед сдачей работы заказчику код проверяется валидатором http://validator.w3.org на соответствие стандартам

Кроссбраузерность

Поддержка сайта в широко используемых браузерах, таких как Chrome, Opera, Firefox, Safari, Microsoft edge, IE9+ и т.д.

Адаптивность

Адаптивная верстка (по желанию заказчика) с использованием фреймворка Bootstrap или Media-Querries.

PerfectPixel

Точная вёрстка по макету. Проверка производится PP плагинами на разрешении равном ширине макета.
